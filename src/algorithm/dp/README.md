# Dynamic Programming

이곳은 DP에 관한 문제를 모아놓은 문서입니다.

### 개념

DP란 큰 문제를 작은 문제의 합으로 푸는 것이다. 다이나믹의 구현 방식에는 두 가지가 있다.

- Top-down: 재귀
- Bottom-up: 반복문

DP를 푸는 과정

1. 테이블 정의하기
2. 점화식 찾기
3. 초기값 정하기
